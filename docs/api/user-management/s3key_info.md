# s3key_info

This is a simple module that supports listing S3Keys.

## Example Syntax


```yaml
name: List s3keys
ionoscloudsdk.ionoscloud.s3key_info:
  user: ''
register: s3key_list_response

```

&nbsp;

&nbsp;

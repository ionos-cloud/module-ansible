# pipeline_info

This is a simple module that supports listing existing Pipelines

## Example Syntax


```yaml

    - name: List Pipelines
        pipeline_info:
        register: pipelines_response

    - name: Show Pipelines
        debug:
            var: pipelines_response.result

```
